How to Use Giveawaybot

Note: This guide assumes you know how to play Giveaway. If not, click here

Giveawaybot is a bot designed to manage the game Giveaway, by assigning random goals, resolving moves, etc. Here are the commands used to interact with Giveawaybot. A "*" indicated that the following command should only be sent in a private message to Giveawaybot, not in the game channel.

Most commands should work in private message, however it is polite to use "join" and "leave" in the channel.

Note: text surrounded by "[]" indicates fields that should be replaced by user input, not sent literally

During Signups

!join -- Adds the user to the pool of players for the next game.

!leave -- Leaves the player pool.

!players -- Lists the current players

During a Normal Move

!hands -- Tells you the current tokens of all players.

*!move [first letter of color to give] [name or color of player to give to] -- Sets your move to be "Give [color] to [player].

For example, "!move g nickthename" would give a green token to the player nickthename. If nickthename started with red, this command could also be written "!move g r"

*!mymove -- Tells you your current move.

During Rebalance

!hands -- Same as above

*!move -- Same as above, but note some players will not be making moves, and some will make multiple, in which case they should issue the command multiple times.

*!mymove -- Same as above

!ready -- During rebalance, all moving players must issue this command before the game proceeds, to ensure no accidental moves. (recommended in private message)
